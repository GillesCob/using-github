Modifications du code

On va prendre l'habitude de travailler en local avec VS Code installé sur notre machine.

Nous n'allons pour le moment pas utiliser de branches.

Etape 1 : Ouvrir son code en local
  • Code => Local => Open with GitHub desktop
  • Ouvrir VS Code depuis GitHub desktop
  • Réaliser les modifications
Etape 2 : Tracker les modifications
  • Regrouper les modifs en paquets et réaliser régulièrement des "Commit"
  • Réaliser des commit séparés pour chaque fichier afin de facilement s'y retrouver
Etape 2 bis : Tracker les modifications
  • git add src/nom_fichier
  • git commit -m "titre concernant ces modifs
  • réitérer autant que nécessaire ces étapes
Etape 3 : Savoir où on en est
  • Pour savoir où on en est : git status
  • Possibilité de vérifier les logs : git log
  • Si on veut plus de détails : git log --stat
